# Dev Ink
A Gui program to create, save and sort different color themes, written in python.

---

## Instalation

### Arch Linux (based)

Stable release:
```shell
yay -S dev_ink
```

Rolling release:
```shell
yay -S dev_ink-git
```

---

### Other OS

Download:
```shell
git clone https://github.com/Veicm/Dev_Ink.git
```

Start:
```shell
cd Dev_Ink
python -m dev_ink
```
