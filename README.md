## Installation

### Arch Linux (based)

Stable release:
```shell
yay -S dev_ink
```

Rolling release:
```shell
yay -S dev_ink-git
```

---

### Other OS

Download:
```shell
git clone https://github.com/Veicm/Dev_Ink.git
```

Start:
```shell
cd Dev_Ink
python -m dev_ink
```

## Usage/Examples

After installing, you can run the program using:
```shell
python -m dev_ink
```
or
```shell
dev_ink
```

To create a new color theme, follow the on-screen instructions.
